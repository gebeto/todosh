# useFocus

This hook is used for handling `focus()` on input ref.
Ref creation and `focus` call is decoupled to the hook.

Simple hook usage.
```js
const Test = () => {
  const [ref, focus] = useFocus();

  return (
    <div>
      <button onClick={focus}>Focus!</button>
      <input ref={ref} />
    </div>
  );
}
```
